RETROQUEST

Site Planning Document

Site Name

Name: RetroQuest

Reasoning: The name combines "Retro" to clearly identify the era of content, and "Quest," which evokes the adventure-heavy nature of classic RPGs and platformers. It suggests a journey of discovery for the user.

Site Purpose

RetroQuest serves as a digital library and discovery tool for gamers interested in the NES, SNES, and Sega Genesis eras. The site provides a curated database of classic titles, technical specifications, and a "wishlist" feature to help users track games they intend to play.

Scenarios

  • "I'm looking for a highly-rated platformer for the SNES that I might have missed in the 90s; where can I find a list?"
  • "I want to save a list of Sega Genesis games so I can remember which ones to look for at the local retro game store."

Color Scheme

The palette is inspired by classic arcade aesthetics and "dark mode" UI to make colorful game art pop.

Primary: #1a1a2e (Backgrounds)
Accent: #f8be00 (Buttons/Links)
Secondary: #0f3460 (Cards/Sections)
Text: #FFFFFF (Headings/Body)

Typography

Heading Font: 'Press Start 2P' (or Monospace)

Used for the logo and main H1/H2 tags to give an authentic "pixel" feel.

Body Font: 'Roboto' (or Sans-Serif)

Used for all descriptions and form labels to ensure high readability and accessibility.
